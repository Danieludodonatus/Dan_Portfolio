PROJECT 01

MentorLink Website

MentorLink is an innovative web application designed to streamline the mentorship experience through intelligent matching and personalized guidance. Built using Firebase Studio AI , the platform offers a seamless onboarding process for mentors and mentees, enabling users to sign up , connect, and collaborate based on skill sets, interests, and professional goals. PROJECT 02

Fixed Deposit Calculator UI

The Fixed Deposit (FD) Calculator is an advanced financial tool designed to assist users in accurately estimating the maturity amount and interest earned from fixed deposit investments. This calculator features a simple, intuitive interface allowing users to input their principal amount, select the deposit tenure, adjust the interest rate, and choose the compounding frequency relevant to their needs. Upon calculation, it provides clear, instant results, detailing both the total maturity value and the interest accrued over the selected period, making it ideal for comparing different investment options and aiding informed financial decisions. Built for accessibility and reliability, As a versatile resource, the FD Calculator empowers users to plan and optimize their savings strategies with confidence. Built with HTML, CSS, JS, and Chart.js.
